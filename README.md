# FedEx Alert Management Dashboard

Human-centered alert dashboard designed in collaboration with FedEx for a Human-Computer Interaction course.

Live Demo: https://mikheil17.github.io/FedEx-Alerts

## What It Is

A front-end implementation of a Figma prototype dashboard focused on improving alert visibility, prioritization, and usability through clear layout and data visualization.

## Tech

HTML | CSS | JavaScript | Figma

## Context

Developed as part of an HCI course at Missisippi State University.
